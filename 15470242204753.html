PHP CI 第一个项目

1.下载项目

从 codeigniter中国 下载最新版本的项目代码。（此时最新版本为3.1.9）

将项目放置到PHP服务器环境内。

2.改名

将文件夹 “CodeIgniter-3.1.9” 改名为 ci。
在浏览器上打开浏览效果。

3.创建Controller

在 application文件夹下的 controllers 文件夹下创建文件：Star.php:

<?php
defined('BASEPATH') OR exit('No direct script access allowed');

class Star extends CI_Controller {
    public function index() {
        echo "欢迎来到CI框架";
    }
}

在浏览器上打开：

4.修改路由器

修改默认index.php 页面

打开 application/config/routes.php 文件，
将最底部的代码：

$route['default_controller'] = 'welcome';

修改为

$route['default_controller'] = 'star';

重新打开浏览器运行看看效果：
